# UI Layer

사용자 인터페이스 구성 요소와 화면을 정의합니다.

## 구조

- `theme/` - Material Design 3 테마
  - `Color.kt` - 색상 정의
  - `Typography.kt` - 타이포그래피
  - `Theme.kt` - 라이트/다크 테마

- `screens/` - 화면 Composables
  - `splash/` - SplashScreen
  - `login/` - LoginScreen
  - `signup/` - SignupScreen
  - `home/` - HomeScreen
  - `transaction/` - TransactionEditScreen

- `components/` - 재사용 가능한 UI 컴포넌트
  - `button/` - 버튼
  - `input/` - 입력 필드
  - `dialog/` - 다이얼로그
